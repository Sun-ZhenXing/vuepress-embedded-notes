# Jetson Nano 工具

  • 1. CPU / GPU 状态工具
  • 2. 摄像头驱动

# 1. CPU / GPU 状态工具

jetson-stats 包含各种 Jetson 状态管理工具，安装：

sudo apt install python3-pip
sudo pip3 install jetson-stats

查看状态：

jtop

显示 Jetson 的状态和所有信息：

jetson_release

查看各种资源：

tegrastats

# 2. 摄像头驱动

如果需要使用摄像头，需要安装驱动：[1]

sudo dpkg -r arducam-nvidia-l4t-kernel
sudo shutdown -r now

或者从脚本安装：

wget https://github.com/ArduCAM/MIPI_Camera/releases/download/v0.0.3/install_full.sh
chmod +x install_full.sh
./install_full.sh -m arducam

通过下面的命令检查所有的视频设备：

ls /dev/vid*

通过下面的命令查看摄像头的内容：

nvgstcapture-1.0 --orientation=2

  1. 在 Jetson Nano 上运行 PyTorch 模型，PyTorch，https://pytorch.org/blog/running-pytorch-models-on-jetson-nano/open in new window ↩︎
